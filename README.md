# 🍳 Chef Claude – AI Recipe Generator

Chef Claude is a **full-stack AI web application** built with **React** on the frontend and **Node.js + Express** on the backend. Users enter a list of ingredients and get a complete, formatted recipe generated by an AI model.

The project started as a frontend-only learning exercise and was later refactored into a **monorepo full-stack architecture** — with a secure backend that protects API keys and handles all AI communication.

---

## 🚀 Features

- Add multiple ingredients dynamically
- Generate recipes using Groq AI (Llama 3.1)
- Vegetarian-safe logic *(recipe stays vegetarian unless non-veg ingredients are detected)*
- Clean Markdown output rendered in the UI
- Secure API key handling via backend (key never exposed to browser)
- Responsive UI

## 🔧 Setup & Run Locally

### Prerequisites
- Node.js v18+
- A free [Groq API key](https://console.groq.com)

### 1. Clone the repository

```bash
git clone https://github.com/vidhisonani/chef-claude-react.git
cd chef-claude-react
```

### 2. Set up the Backend

```bash
cd backend
npm install
```

Create a `.env` file inside `backend/`:
```env
GROQ_API_KEY=your_groq_api_key_here
FRONTEND_URL=http://localhost:5173
```

Start the backend server:
```bash
npm run dev
# Server runs on http://localhost:5000
```

### 3. Set up the Frontend

Open a new terminal:
```bash
cd frontend
npm install
```

Create a `.env` file inside `frontend/`:
```env
VITE_API_URL=http://localhost:5000
```

Start the frontend:
```bash
npm run dev
# App runs on http://localhost:5173
```
